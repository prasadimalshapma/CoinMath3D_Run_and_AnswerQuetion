# CoinMath3D_Run_and_AnswerQuetion

CoinMath3D is an educational Unity game built with C# to make learning mathematics more engaging and fun. Players collect coins, face enemies and complete missions by answering MCQ math questions correctly.

Gameplay Overview:
-The game currently includes three levels.
-To level up , players must reach the required marks threshold.
-Marks are earned by:
   -Defeating enemies.
   -Answering math MCQs correctly.
-Wrong answers or missed missions reduce progress, encouraging accuracy and focus.

Technology Stack:
-Engine  : Unity
-Language: C#
-Platform: PC (expandable to mobile in future)

Educational Focus:
This project is designed to:
-Help students practice math in a stress-free , gamified environment.
-Build confidence through interactive challenges.
-Combine fun gameplay mechanics with academic learning goals.

Future Improvments:
-Add more levels with increasing difficulty.
-Introduce time based challenges for extra excitement.
-Multiplayer or leaderboard system for competitive learning.
-Enhanced visuals and sound effects for immersive experience.
-Support for mobile device(Android/ios)
